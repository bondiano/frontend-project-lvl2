# Hexlet frontend project Diffgen

![](https://github.com/bondiano/frontend-project-lvl2/workflows/Node%20CI/badge.svg)

## Installation

Install all dependencies:
```
make install
```
Create package:
```
make publish
```
Install packages:
```
npm link
```
